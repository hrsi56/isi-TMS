01 · THE PROBLEM

TMS therapy is a one-size-fits-all protocol — but brains aren't.

Paired Associative Stimulation (PAS) drives brain plasticity by firing two cortical sites at a precise interval (IPI). The therapeutic window is narrow — milliseconds matter.

Current practice: use one fixed IPI (typically 8 ms) across all patients. Clinical response varies widely.

THE ML QUESTION

Can we extract a per-subject feature from raw EEG that predicts their optimal stimulation parameter?

  • Input: multi-channel resting-state EEG (5 min, 256 Hz)
  • Output: scalar — predicted optimal IPI ∈ {4, 8, 16, 32 ms}
  • Challenge: N=24. Deep learning is out. Feature engineering is in.

EXPERIMENTAL DESIGN

Each session: resting-state EEG → PAS stimulation (one condition) → resting-state EEG. The post-minus-pre change in alpha-band power defines the PAS-induced effect. The condition producing the largest per-subject effect defines the individual optimal IPI — our prediction target.

05 · THE CORE METHOD

Phase-lag as a physical measure of signal propagation.

THE IDEA

Two electrodes watching the same oscillating cortical signal see it at slightly different times. The phase difference between them — at a given frequency — is proportional to how long the signal took to travel across the brain.

prop_time = (1000/f) × (Δφ/2π)

where Δφ = angle(E₂) − angle(E₁) at frequency f

DESIGN CHOICE

Averaged across 13–16 Hz (high-alpha/low-beta). Chosen empirically: narrow band for phase stability, away from mu-rhythm artifacts.

IMPLEMENTATION (MATLAB)

% Phase difference across hemispheres
phaseDiff = angle(F4) - angle(F3);
phaseDiff_abs = abs(phaseDiff);

% Average across 13-16 Hz band
band = freqs > 13 & freqs < 16;
phaseDiff_band = mean(phaseDiff_abs(:,band), 2);

% Convert to propagation time (ms)
propTime = (1000 ./ freqs(band)) .* ...
           (phaseDiff_band ./ (2 * pi));

% Per-subject estimate
subjectFeature = mean(propTime, 'all');

06 · RESULTS

The feature predicts what we hoped.

The extracted per-subject phase-lag correlates significantly with the individually optimal IPI — subjects with longer inter-hemispheric propagation times need longer inter-pulse intervals to maximize stimulation response.

r = 0.49
p = 0.015 · linear regression, 2-tailed
Right hemisphere stimulation site (F4 electrode)

07 · VALIDATION

Not an artifact — the asymmetry is meaningful.

RIGHT HEMISPHERE (F4)

r = 0.49
p = 0.015
Significant

Phase-lag at this site tracks the true conduction delay for the stimulated circuit.

LEFT HEMISPHERE (F3)

r = 0.04
p = 0.85
Not significant

Left-hemisphere phase-lag is dominated by other dynamics — the biomarker is site-specific, as expected biologically.

Why this matters: the control direction is null, which rules out the main confound that the correlation is driven by some subject-level nuisance variable.

09 · WHAT THIS ENABLES

From a fixed protocol to subject-tailored stimulation.

Pre-session EEG → IPI recommendation

A 5-minute resting-state EEG — collected before the first therapy session — is enough to extract the biomarker and select an IPI from the discrete set {4, 8, 16, 32 ms}. No additional hardware, no behavioral task.

Who is unlikely to respond

Subjects whose propagation time falls outside the range where any discrete IPI provides a good match are flagged as likely non-responders — information a clinician can act on before committing to a multi-week course.

A motivated next study

An observational correlation is not a clinical result. The natural follow-up is a randomized controlled trial with stimulation individually adjusted from the biomarker, versus the current fixed protocol.
